# MLRP-KG: Mine Landslide Risk Prediction

This project aims to predict mine landslide risks using a knowledge graph approach. The project follows weekly tasks where Kaggle datasets are used to develop models and report the results.

---

## Objective

- Predict landslide risks in mining areas  
- Apply knowledge graph-based methods  
- Develop skills in data analysis, model development, and reporting

---

## Datasets

- Kaggle: Landslide Dataset ([https://www.kaggle.com/datasets/rajumavinmar/landslide-dataset))  

> Note: Please follow the licensing rules for each dataset.

---

## Project Structure

- `notebooks/` : Weekly Jupyter Notebooks for tasks  
- `reports/` : Weekly reports and visualizations  
- `README.md` : Project description and usage instructions
